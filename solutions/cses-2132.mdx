**Time Complexity**: $\mathcal O(N\log N)$

This problem is similar to
[CF 713C](https://codeforces.com/contest/713/problem/C) except that we're
supposed to create a **non-decreasing array**.

Here is [zscoder's tutorial](https://codeforces.com/blog/entry/47821).

<LanguageSection>

<CPPSection>

```cpp
#include <stdio.h>
#include <queue>
using namespace std;

int n, t;
long long ans = 0;
priority_queue<int> Q;

int main() {
	scanf("%d", &n);
	for(int i=0; i<n; i++) {
		scanf("%d", &t);
		Q.push(t);
		ans += Q.top() - t;
		Q.pop(); Q.push(t);
	}
	printf("%lld", ans);
	return 0;
}
```

<PySection>

```py 
from heapq import heappop, heappush, heapify
priority_queue = []
heapify(priority_queue)
ans=0
n=int(input())
for _ in range(n):
	t=int(input())
	heappush(priority_queue,-1*t)
	ans+=(priority_queue[0]*-1)-t
	heappop(priority_queue)
	heappush(priority_queue,-1*t)
print(ans)

```
